## ⚠️ Риски и безопасность (отдельный блок)

### 🔴 Критические риски и стратегии минимизации

| Риск | Вероятность | Последствия | Стратегия снижения |
|------|-------------|-------------|-------------------|
| **Поломка праворульного авто в глуши** | Средняя | Задержка 1–3 дня, дорогой ремонт | ✅ Взять набор запчастей (лампы, предохранители, ремень), заранее найти СТО в крупных городах, иметь контакты эвакуаторов |
| **Отсутствие связи (Забайкалье, север Иркутской обл.)** | Высокая | Потеря навигации, невозможность вызвать помощь | ✅ Скачать офлайн-карты (Яндекс, Maps.me), купить бумажную атлас, взять рации (2 шт.), сообщать маршрут родным |
| **Усталость водителя / микросон** | Высокая | Авария, травмы | ✅ Смена за рулём каждые 2–3 ч, обязательный сон 7+ ч, кофе/энергетики только как доп., а не замена сна |

| **Перепады температур (−3° ночью в Забайкалье, +35° днём)** | Средняя | Переохлаждение/перегрев, поломка авто | ✅ Одежда слоями, термобельё, одеяло в машине, проверка кондиционера и печки перед выездом |
| **Кража/вандализм на стоянках** | Низкая | Потеря вещей, стресс | ✅ Ночёвки только на охраняемых АЗС/парковках, ценные вещи убирать в багажник, не оставлять документы в салоне |
| **Проблемы с документами/ОСАГО** | Низкая | Штрафы, задержание | ✅ Распечатать ОСАГО + СТС, проверить срок действия, иметь цифровые копии в телефоне и облаке |
| **ДТП с фурой / сложный рельеф (Байкал, серпантины)** | Средняя | Травмы, простой, ремонт | ✅ Дистанция 3+ сек, не обгонять на подъёмах, дневной свет фар, видеорегистратор (2 камеры) |

## 🔧 Подготовка автомобиля: чек-лист за 1 неделю до выезда

- [ ] Полное ТО: замена масла, фильтров (воздушный, салонный), проверка тормозов
- [ ] Диагностика подвески, рулевого управления, ШРУСов
- [ ] Проверка уровня и состояния: антифриз, тормозная жидкость, масло в АКПП/вариаторе
- [ ] Шины: летние, протектор >4 мм, давление по норме (включая запаску)
- [ ] Аккумулятор: проверить заряд, очистить клеммы (для гибрида — и вспомогательный АКБ)
- [ ] Свет: все фары, поворотники, стоп-сигналы, габариты (правый руль — лампы могут быть нестандартные)
- [ ] Кондиционер: проверить работу (летом в Забайкалье +35° — критично)
- [ ] Стеклоочистители: заменить щётки, залить омывайку
- [ ] Документы: проверить срок ОСАГО, СТС, прав
- [ ] Тест-драйв: проехать 100–200 км с полной загрузкой, послушать посторонние шумы

# ⚠️ Оценка безопасности и анализ рисков
## Честный вердикт перед стартом

**Краткий ответ:** Отправляться в поездку **немедленно после получения автомобиля** с такими вводными **категорически небезопасно**. Совокупность факторов (10-летний перерыв в вождении, отсутствие опыта на механике, нулевой опыт на праворульном авто, нулевая акклиматизация к конкретной машине + 9200 км трассы) создаёт **критический уровень риска**. Статистика ДТП с участием водителей после длительного перерыва и на незнакомой компоновке руля показывает кратное возрастание вероятности ошибок в первые 500–1500 км.

Однако, если друг **твердо намерен ехать**, риски можно снизить до приемлемого уровня только через **обязательный подготовительный этап**. Ниже — детальный разбор, план минимизации и байесовская модель оценки вероятности успеха.

## 📐 Байесовский расчёт вероятности успеха

### 🔢 Формула и логика
Байесовский вывод позволяет **последовательно обновлять вероятность** события при появлении новых фактов:
```
P(S|E) = [P(E|S) × P(S)] / P(E)
```
Где:
- `S` = успешное завершение поездки (без ДТП, поломок >24ч, с сохранением здоровья и бюджета)
- `E` = совокупность известных фактов
- `P(S)` = априорная вероятность (базовый уровень до учёта деталей)
- `P(E|S)` = вероятность увидеть эти факты, если поездка успешна
- `P(E)` = общая вероятность фактов

Для практического расчёта удобнее использовать **форму отношения шансов (Odds)**:
```
Odds_новое = Odds_старое × LR
P = Odds / (1 + Odds)
```
где `LR` (Likelihood Ratio) = `P(фактор|Успех) / P(фактор|Неудача)`

> ⚠️ Значения LR оценочные, основаны на статистике ДТП, исследованиях деградации навыков вождения и отчётах страховых компаний. Это **инструмент осознанного принятия решений**, не точный прогноз.

### 📊 Пошаговое обновление вероятности

| Факт (E) | LR | Odds до | Odds после | Вероятность P |
|----------|----|---------|------------|---------------|
| **Базовый уровень** (опытный водитель, знакомое авто, плановый рейс) | 1.0 | 1.33 | 1.33 | **57%** |
| 🔻 10 лет без практики | 0.45 | 1.33 | 0.60 | 37% |
| 🔻 Только «пару занятий» на МКПП | 0.50 | 0.60 | 0.30 | 23% |
| 🔻 Нулевой опыт на праворульном авто | 0.60 | 0.30 | 0.18 | 15% |
| 🔻 Авто только из Японии, без обкатки | 0.70 | 0.18 | 0.13 | 11% |
| 🔻 Сразу 9200 км без акклиматизации | 0.55 | 0.13 | 0.07 | 6.5% |
| 🔺 Второй водитель опытный, готов брать 60% | 1.50 | 0.07 | 0.11 | 9.5% |
| 🔺 Внедрение плана подготовки (2–4 нед + тест-рейс) | 2.20 | 0.11 | 0.24 | **19%** |
| 🔺 Жёсткий режим: смена каждые 2ч, макс 400 км/день первые 3 дня, резерв 30% | 1.80 | 0.24 | 0.43 | **30%** |

📌 **Итог без подготовки:** ~6–10%  
📌 **Итог со строгой подготовкой:** ~30–35%  
📌 **Приемлемый порог для старта:** ≥60% (требует 4–6 нед практики + второго опытного водителя, знакомого с RHD)

## 🧠 Как считать в уме (практический метод)

Для быстрой оценки в дороге или при обсуждении используйте **метод корректировок базовой вероятности**:

1. **Старт:** Возьмите `P₀ = 0.70` (успешность рейса при штатных условиях)
2. **Каждый фактор-риск** умножайте на коэффициент `k` (0.5–0.9)
3. **Каждый фактор-защита** умножайте на `k` (1.1–1.3)
4. **Не выходите за границы 0.10–0.95**
5. **Пример расчёта:**
   ```
   P = 0.70 
     × 0.5 (10 лет без практики) 
     × 0.6 (МКПП неуверенно) 
     × 0.7 (нет опыта RHD) 
     × 1.3 (второй водитель опытный) 
     × 1.2 (будет обкатка 500 км) 
     × 0.9 (смена каждые 2ч) 
   P ≈ 0.70 × 0.5 × 0.6 × 0.7 × 1.3 × 1.2 × 0.9 ≈ 0.19 → 19%
   ```
6. **Правило принятия решения:**
   - `< 30%` → ❌ Не ехать. Нужна подготовка.
   - `30–50%` → ⚠️ Ехать только при готовности к задержкам, дополнительным расходам и строгом соблюдении режима.
   - `> 60%` → ✅ Можно стартовать.
